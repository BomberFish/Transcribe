# Transcribe

A work-in-progress web application to transfer playlists between music streaming services. Interop or die!

## Supported Services

- Apple Music
- YouTube Music

## What works
- Enumerating playlists:
  - Apple Music
  - YouTube Music
- Transferring:
  - Apple Music <-> YouTube Music
